== Description ==

Tagesgeld - Aktuelle Tagesgeldzinsen werden in der Sidebar angezeigt. Konfigurierbar &uuml;ber ein Widget. [http://www.tagesgeld.org/](http://www.tagesgeld.org/)

Das Wordpress-Template muss widget-f&auml;hig sein. Anleitung dazu: [http://automattic.com/code/widgets/themes/](http://automattic.com/code/widgets/themes/)

*Feature List*

* customizable widget
* displays a user-defined number of banks

== Installation ==

1. Extract `tagesgeld.zip`.
2. Upload the `tagesgeld`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.

*German*
a. Entpacke `tagesgeld.zip`.
b. Lade den `tagesgeld`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
c. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
d. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
